# Angular 2 快速指南

* 创建 package.json；
* 创建 tsconfig.json；
* 创建 typings.json；
* 创建 webpack.config.js；
* 安装依赖 `npm install` (注意：postinstall 命令会在安装完依赖后继续安装)；
* 全局安装 typings ：`$(npm bin)/typings install --global --save dt~core-js`；
* 创建应用文件 src/
    * 创建首页 src/index.html
    * 创建入口程序 src/main.ts
    * 创建模块 src/app/hello-world.ts
* 编译 `npm run build`
* 启动服务 `npm run serve`
